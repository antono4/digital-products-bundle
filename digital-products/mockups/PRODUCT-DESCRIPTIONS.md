## 📋 TIPS PENULISAN DESKRIPSI

### Do's:
✓ Start with benefit, not feature
✓ Use emojis untuk visual appeal
✓ Include "what's included" list
✓ Mention target audience
✓ Add social proof elements
✓ Include clear CTA

### Don'ts:
✗ Jangan terlalu panjang (potong jadi beberapa paragraph)
✗ Jangan pakai jargon yang rumit
✗ Jangan promise yang tidak bisa ditepati
✗ Jangan lupa mention format file
✗ Jangan lupa mention instant download
