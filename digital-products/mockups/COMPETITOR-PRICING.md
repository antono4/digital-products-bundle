## 🎯 STRATEGI PRICING

### 1. Penetration Pricing (Awalan)
```
✓ Harga 20-30% di bawah kompetitor
✓ Tujuan: build reviews & establish presence
✓ Durasi: 30-60 hari pertama
✓ Setelah 10+ reviews → naikkan harga
```

### 2. Skimming Pricing (Premium)
```
✓ Harga 10-20% di atas kompetitor
✓ Gunakan jika produk sangat premium
✓ Fokus pada perceived value
✓ Pastikan mockup & deskripsi premium
```

### 3. Bundle Pricing
```
✓ Sell individual: $10-15 each
✓ Sell bundle all 5: $45-55 (save 20-30%)
✓ Tujuan: increase average order value
```

### 4. Seasonal Pricing
```
✓ January: New Year resolution → Planner boost
✓ March-April: Spring cleaning → Minimalism boost
✓ September: Back to school → Social kit boost
✓ November-December: Holiday → Gift-ready items
```

## 💡 TIPS OPTIMASI HARGA

### Do's:
✅ Start dengan harga yang sedikit rendah untuk build momentum
✅ Gunakan psychological pricing ($18.99, $22.97)
✅ Offer bundle discounts (10-20% off)
✅ Gunakan "reguler price" untuk show discount
✅ Test different price points after 30 days

### Don'ts:
❌ Jangan terlalu murah (undervalue produk)
❌ Jangan terlalu mahal di awal (belum ada social proof)
❌ Jangan ubah harga drastis (bisa confuse customers)
❌ Jangan compete hanya di harga (focus on value)
