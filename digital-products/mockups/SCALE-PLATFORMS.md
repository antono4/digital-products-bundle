## Platform 2: ETSY

### Kenapa Etsy?

```
✅ KEUNTUNGAN:
- Traffic organik tinggi
- Marketplace established
- Customer sudah ready to buy
-SEO built-in

❌ KEKURANGAN:
- Biaya: $0.20 per listing + 6.5% transaction
- Competition tinggi
- Perlu optimization yang baik
```

### Langkah Setup Etsy:

**1. Buat Akun:**
```
→ Buka Etsy.com/signup
→ Pilih "Sell on Etsy"
→ Setup shop preferences
```

**2. Setup Shop:**
```

├── Shop name: [BrandName]Printables atau [Name]Designs
├── Shop language: English
├── Country: [Country]
├── Currency: USD
└── Product type: Digital Items

🏪 Shop Policies:
├── Processing time: 0-1 days (digital)
├── Refund policy: No refunds (digital)
└── Copyright: I can confirm
```

**3. Listing Products:**

