## Langkah 4: Cara Compress Files

### Di Mac/Windows:
1. Select semua file dalam folder produk
2. Right-click → "Compress" (Mac) atau "Send to > Compressed folder" (Windows)
3. Rename jadi: `[nama-produk].zip`

### Di Linux:
```bash
cd /workspace/project/digital-products
zip -r canva-social-kit.zip canva-social-kit/
```

## Langkah 5: Upload Thumbnail/Mockup

### Cara Buat PNG dari HTML Mockup:

**Opsi 1: Screenshot Manual**
1. Open file HTML mockup di browser
2. Screenshot entire page
3. Crop jadi aspect ratio: **3:2** (1200x800 atau 600x400)

**Opsi 2: Gunakan Canva**
1. Buka Canva.com
2. Create design: Custom size (1200 x 800 px)
3. Import screenshot mockup
4. Add text overlay dengan product name
5. Download sebagai PNG

**Opsi 3: Figma (Free)**
1. Buka Figma.com
2. Import HTML file
3. Export sebagai PNG

### Thumbnail Requirements:
```
📐 Ukuran ideal: 1200 x 800 px (3:2 ratio)
📁 Format: JPG atau PNG
📊 Max size: 5MB
💡 Tips: Use striking colors, clear text, product preview
```

## Langkah 6: Product Page Optimization

### Title Best Practices:
- ✅ Clear & descriptive
- ✅ Include main keywords
- ✅ Under 60 characters
- ✅ Benefit-driven

### Good Examples:
```
✅ "Complete Instagram Branding Kit - 30+ Canva Templates"
✅ "30-Day Minimalism Challenge Workbook - Transform Your Life"
❌ "Social Media Kit v2"
❌ "My Templates Pack"
```

### Description Structure:
```markdown
## [Hook - 1 sentence]

[Problem statement - 1-2 sentences]

### What You Get:
- Item 1
- Item 2
- Item 3

### Why This Product:
[Brief value proposition]

---

**Price: $XX**
[Buy Button]
```

### Tags Tips:
```
✓ Use 8-10 relevant tags
✓ Mix broad + specific tags
✓ Include:
  - Product type (template, ebook, svg)
  - Platform (canva, cricut)
  - Industry (social media, business)
  - Use case (branding, planning)
  - Audience (freelancer, creator)
```

❌ Jangan skip thumbnail (akan pakai default)
